## ✨ 新增功能

### 按键气泡提示 (Key Hint Overlay)

在猫咪头顶实时显示当前按下的按键，适合直播展示、快捷键教学等场景。

- 自定义垂直位置（0-50%）
- 自定义字体大小（8-48px）
- 7 种文字颜色可选
- 5 种背景透明度可选（含纯透明无背景模式）
- 可选白色光晕边框
- 自动格式化按键名称（去掉 `Key`/`Digit`/`Num` 前缀，修饰键显示符号）

**使用方式**：打开偏好设置 → 猫咪设置 → 按键提示设置
